Spring Boot + PostgreSQL + Vault で構築した勤怠管理アプリです。

## 🛠️ 技術スタック

| 技術 | 用途 |
|---|---|
| Spring Boot 3.3 | バックエンドAPI |
| PostgreSQL 15 | データベース |
| HashiCorp Vault | シークレット管理 |
| Docker Compose | コンテナ管理 |
| Spring Security | 認証・認可 |

## 🚀 起動方法

```bash
docker-compose up -d
```

ブラウザで http://localhost:8080 を開く

## 📦 機能

- ✅ 出勤打刻
- ✅ 退勤打刻
- ✅ 勤怠履歴一覧
- ✅ 今月の出勤日数・総勤務時間・残業時間
- ✅ ログイン認証
